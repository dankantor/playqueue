Class: PlayQueue

PlayQueue

new PlayQueue(opts)

Create a PlayQueue

Parameters:
Name Type Description
opts Object
Properties
Name Type Argument Default Description
audio Audio

The underlying audio object.
limit number <optional>
 -1

If set will limit the max length of the list
loadTimeout number <optional>
 15000

Number of milliseconds to wait for a song to load before moving on to next song
useLocalStorage boolean <optional>
 false

If true will save list, position, and shuffle state in localStorage. Subsequent page loads will initialize with these saved values.
localStorageNS string <optional>
 'playqueue'

If useLocalStorage is true, items will be saved with the default namespace 'playqueue'. You can change the namespace to a different value.
Example
import {PlayQueue} from 'playqueue'; 

const audio = new Audio();

const playQueue = new PlayQueue({
  'audio': audio,
  'limit': 200,
  'useLocalStorage': true,
  'localStorageNS': 'myapp'
});

Methods

add(params)

Add one or more Songs to the list.

Parameters:
Name Type Description
params PlayQueue~Song | Array.<PlayQueue~Song> | string

Song object or array. If string, pass in url of the song.
Throws:

Added songs cannot exceed limit if set.

Type
RangeError
Examples
// add multiple songs
playQueue.add([
  {'url': '1.mp3', 'title': 'One', 'artist': 'Cool Band', 'album': 'The Hits'},
  {'url': '2.mp3', 'title': 'Two', 'artist': 'Cool Band', 'album': 'The Hits'}
]);
// add a single song
playQueue.add({'url': '1.mp3', 'title': 'One', 'artist': 'Cool Band', 'album': 'The Hits'});
// add a single song with just a url
playQueue.add('1.mp3');
